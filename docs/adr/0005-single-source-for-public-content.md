# ADR-0005: 全部公开语义内容采用唯一权威来源

两个网站实现需要表达同一个作者身份和同一组公开内容。分别维护文章、项目、About、Contact、标签、路径和媒体会导致含义漂移，也会阻碍 Pages 版未来升级为灾备站。因此，全部公开语义内容只能在一个权威内容源中创作和修改，再由两个实现消费或派生。

两版可以保留不同的布局、动画、搜索、图片处理、构建和部署方式；这些实现差异不得形成第二份可独立修改的公开语义内容。权威内容源的存放位置由 ADR-0006 决定，同步方式另行决策。

## Consequences

- 作者身份、联系方式、文章、项目、页面文案、标签、公开路径和媒体采用统一定义。
- 两版需要建立内容适配层，以兼容现有不同目录和 frontmatter 字段。
- Pages 版即使暂不参与搜索索引，也应保持与主站一致的已发布内容。
- 内容一致性需要成为自动验收条件，而不能依赖人工记忆同步。

## Alternatives Considered

**只共享文章**

- 未采用：项目、About、Contact、标签和路径仍会形成品牌与语义漂移。

**两版继续分别维护全部内容**

- 未采用：重复编辑不可持续，也无法为灾备接管提供可信的一致性基础。
