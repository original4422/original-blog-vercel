# 个人数字花园

本项目是承载个人身份、项目作品与长期写作内容的统一网站，而不是单一用途的文章站点或作品集。

## Language

**个人数字花园**:
融合个人主页、项目作品集与技术博客的个人网站，三类内容共同构成作者在网络上的长期公开表达。
_Avoid_: 纯博客、纯作品集

**作者**:
网站对外展示的唯一创作者身份，公开名称为 `original`。
_Avoid_: Yun Shen、站长、管理员

**主站**:
承载唯一正式品牌入口、规范网址与搜索索引权威的个人数字花园。
_Avoid_: 正式版、Vercel 版

**对照站**:
公开展示另一种技术路线、供直接访问和验证，但不承担品牌入口、搜索索引权威或故障接管承诺的网站。
_Avoid_: 第二主站、备站、灾备站

**灾备站**:
与主站保持可验证的内容同步，并具备明确域名切换机制和恢复目标、能够在主站故障时接管访问的网站。
_Avoid_: 对照站、镜像站、备份页面

**公开语义内容**:
决定作者对外表达含义的全部公开信息，包括身份、联系方式、文章、项目、页面文案、标签、公开路径和媒体；其含义不因网站实现不同而改变。
_Avoid_: 页面数据、Vercel 内容、Pages 内容

**权威内容源**:
公开语义内容唯一允许被创作和修改的来源，其他网站实现只能从它派生内容。
_Avoid_: 主站内容、内容副本、同步目录

**温备站**:
持续使用主站同源内容构建和部署、能够在人工操作后接管访问，但尚未完成自动切换或完整恢复验证的网站。
_Avoid_: 对照站、热备站、已验证灾备站

**异地 Git 镜像**:
保存在权威源码平台故障域之外、用于恢复完整源码和历史的只读 Git 副本，不参与日常创作或发布决策。
_Avoid_: 第二内容源、开发仓库、同步分支

**生产意图**:
`main` 分支中两套应用应当自动收敛到的最新可发布状态；它表达期望状态，不证明任一平台已经完成部署。
_Avoid_: 候选分支、已上线版本、Vercel Production

**线上版本**:
某个托管平台当前实际提供访问的源提交，由提交标识验证；主站与温备站可以在部署窗口内短暂不同。
_Avoid_: main、最新代码、生产意图
